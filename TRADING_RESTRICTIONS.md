# Trading restrictions (EA_SCALPER_XAUUSD_CORE)

This repository is published for **study, analysis, and demonstration only**.

## Forbidden uses

You **may not** use any part of this repository (source code, binaries, configuration, or derived works) for any of the following:

- **Live trading** (real-money accounts)
- **Funded / prop trading** (evaluations or funded accounts)
- **Any paper trading that is connected to a broker or routes orders** (even if it uses “demo” money)
- **Any use that mirrors live execution** (e.g., VPS + broker connection, any real-time order routing)
- **Commercial use** (selling, licensing, offering as a service, paid signals/copy, monetized products)

These forbidden uses apply even if you do not charge money and even if you call it “testing” or “paper trading”.

## Allowed uses

- Reading the code to learn
- Static analysis and experimentation in a local sandbox
- Backtesting in a simulated environment **without** connecting to a broker and **without** placing real orders
